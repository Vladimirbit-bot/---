## 1. 🗺️ Навигация и построение маршрутов

> **Задача:** Построить оптимальный маршрут для прогулки с учётом
> препятствий, дорог, парков, погоды и особенностей питомца.

| Задача | Технология / Библиотека | Почему именно она |
|---|---|---|
| Построение графа дорог и троп | **OSMnx + NetworkX** | OSMnx загружает реальные карты из OpenStreetMap и строит граф улиц; NetworkX работает с этим графом для поиска путей. Связка бесплатная и хорошо документированная |
| Поиск кратчайшего / оптимального пути | **Алгоритм A\*** (реализация через NetworkX) | A\* — классика навигации: учитывает не только длину пути, но и «стоимость» каждого участка (шумная дорога = высокая стоимость). Быстрее Дейкстры на больших графах |
| Динамическое избегание препятствий в реальном времени | **D\* Lite** (динамический A\*) | В отличие от статического A\*, D\* Lite пересчитывает маршрут «на лету» при появлении новых препятствий (лужа, стройка, агрессивная собака). Используется в реальной робототехнике |
| Навигация робота в пространстве (SLAM) | **ROS 2 + Nav2 + Cartographer** | ROS 2 (Robot Operating System) — стандартный фреймворк для робототехники. Nav2 — стек автономной навигации. Cartographer строит карту помещения/улицы в реальном времени |
| Учёт внешних факторов (погода, загруженность) | **OpenWeatherMap API + Google Maps API** | OpenWeatherMap даёт прогноз по геолокации (дождь → избегаем маршруты без навесов). Google Maps API — данные о загруженности и типах дорог |
| Персонализация маршрута под питомца | **Scikit-learn (Decision Tree / Random Forest)** | ML-модель учитывает породу, возраст, вес, болезни собаки и выбирает подходящую интенсивность и длину маршрута на основе обучающих данных |
| Геозонирование (виртуальный забор) | **Shapely + GeoPandas** | Shapely позволяет задать полигон безопасной зоны и мгновенно проверять, находится ли питомец внутри. GeoPandas — для работы с геоданными |

### 💡 Как это работает вместе

```
[Данные о собаке] ──► [ML-персонализация]
                              │
[OpenStreetMap]  ──► [OSMnx граф] ──► [A* / D* Lite] ──► [Маршрут]
                              │                               │
[Погода, трафик] ─────────────┘              [ROS 2 Nav2] ◄─┘
                                                    │
                              [Робот едет по маршруту]
```

## 2. 👁️ Компьютерное зрение

> **Задача:** Распознавать агрессивных собак, лужи, препятствия,
> опасные объекты на улице в реальном времени с камеры робота.

| Задача | Технология / Библиотека | Почему именно она |
|---|---|---|
| Базовое распознавание объектов (собаки, люди, препятствия) | **YOLOv8 (Ultralytics)** | YOLO — самая быстрая модель детекции объектов в реальном времени. YOLOv8 работает на 30+ FPS даже на слабом железе. Есть готовые веса для животных |
| Распознавание луж и мокрых поверхностей | **Сегментация: YOLOv8-seg / SAM (Segment Anything Model)** | SAM от Meta умеет сегментировать любые объекты без дополнительного обучения. Для луж — обучаем YOLOv8-seg на собственном датасете с разметкой поверхностей |
| Оценка агрессивного поведения собаки | **MediaPipe Pose + кастомный классификатор** | MediaPipe определяет позу и скелет животного (поднятая шерсть, напряжённое тело, оскал). Поверх — обучаем бинарный классификатор (агрессия / не агрессия) на датасете поведения собак |
| Глубина сцены (расстояние до препятствий) | **Intel RealSense D435 + Open3D** | Стереокамера RealSense даёт карту глубины в реальном времени. Open3D обрабатывает облако точек для построения 3D-карты препятствий |
| Ночное зрение / плохое освещение | **Тепловизор FLIR Lepton + OpenCV** | FLIR Lepton — компактный тепловизионный модуль. OpenCV объединяет тепловое изображение с RGB для работы в темноте |
| Распознавание породы собаки | **EfficientNet (fine-tuned на Stanford Dogs Dataset)** | Stanford Dogs Dataset — 20 580 фото 120 пород. EfficientNet — лёгкая и точная свёрточная сеть, идеально для классификации пород |
| Трекинг объектов между кадрами | **DeepSORT** | DeepSORT назначает уникальный ID каждому объекту и отслеживает его движение между кадрами. Нужен, чтобы не «терять» агрессивную собаку при перекрытии кадра |

### 💡 Пайплайн компьютерного зрения

```
[Камера робота]
      │
      ▼
[YOLOv8 — детекция объектов]
      │
      ├──► Собака? ──► [Классификатор агрессии (MediaPipe + ML)]
      │                        │
      │                 Агрессивная? ──► [Сигнал опасности] ──► [D* Lite: объезд]
      │
      ├──► Лужа? ──► [SAM сегментация] ──► [Отметить на карте] ──► [Объезд]
      │
      └──► Препятствие? ──► [RealSense глубина] ──► [Nav2: торможение]
```

## 3. 🎬 Генерация видео для TikTok

> **Задача:** Автоматически создавать короткий монтажный ролик
> из записи прогулки — с музыкой, титрами и эффектами.

| Задача | Технология / Библиотека | Почему именно она |
|---|---|---|
| Выбор лучших моментов из видео (highlight detection) | **PySceneDetect + OpenCV** | PySceneDetect автоматически находит смену сцен и динамичные моменты. OpenCV анализирует яркость, движение и «интересность» кадра |
| Автоматический монтаж и склейка клипов | **MoviePy** | Python-библиотека для программного видеомонтажа: склейка клипов, наложение музыки, добавление титров, переходы — всё через код |
| Генерация субтитров и текстовых оверлеев | **Whisper (OpenAI) + Pillow** | Whisper транскрибирует любые звуки (лай, команды хозяину). Pillow рендерит красивые титры поверх видео |
| Подбор музыки под настроение прогулки | **Essentia (музыкальный ИИ) + библиотека Epidemic Sound API** | Essentia анализирует темп и настроение видео. Epidemic Sound API подбирает лицензионный трек без авторских прав |
| Генерация коротких вертикальных видео (формат TikTok/Reels) | **FFmpeg + кастомные шаблоны** | FFmpeg — стандарт видеообработки. Конвертирует в формат 9:16, добавляет рамки, логотип и watermark |
| AI-улучшение качества видео | **Real-ESRGAN** | Нейросеть для апскейлинга и улучшения качества видео. Превращает мутные кадры с камеры робота в чёткие 4K-кадры |
| Генерация полностью синтетических видео (будущее) | **Runway ML Gen-2 / Sora API (когда откроют)** | Runway Gen-2 уже сейчас умеет генерировать короткие видеоклипы по текстовому промпту. Можно сгенерировать «идеальный» момент прогулки |

### 💡 Пайплайн генерации TikTok-ролика

```
[Видео с камеры робота (30 мин)]
            │
            ▼
[PySceneDetect — поиск лучших моментов]
            │
            ▼
[Real-ESRGAN — улучшение качества кадров]
            │
            ▼
[MoviePy — монтаж топ-10 моментов в 30-секундный ролик]
            │
            ├──► [Whisper — субтитры]
            ├──► [Essentia — подбор музыки]
            └──► [Pillow — титры и логотип]
                        │
                        ▼
            [FFmpeg — конвертация в 9:16]
                        │
                        ▼
              [Готовый TikTok-ролик 🎬]
```

## 4. 💬 LLM и генерация текста (VIP-тариф)

> **Задача:** Генерировать персонализированные тексты от лица собаки —
> отчёты о прогулке, посты в соцсети, а также торжественные речи
> для собак на VIP-тарифе (да, это реальная фича 🐾).

| Задача | Технология / Библиотека | Почему именно она |
|---|---|---|
| Генерация текстового отчёта о прогулке | **GPT-4o (OpenAI API)** | Лучшая коммерческая LLM для генерации связных, персонализированных текстов. Принимает данные прогулки (маршрут, активность, фото) и пишет отчёт «от лица собаки» |
| Локальная LLM (если нет интернета или бюджета) | **Llama 3 (Meta) через Ollama** | Llama 3 — мощная открытая модель, работает локально на сервере без облака. Ollama упрощает запуск до одной команды в терминале |
| RAG (контекст о конкретной собаке) | **LangChain + ChromaDB** | LangChain управляет цепочками запросов к LLM. ChromaDB хранит векторизованные данные о питомце (имя, порода, привычки, любимые места) — LLM всегда «помнит» собаку |
| Генерация торжественных речей для собак (VIP) | **GPT-4o с кастомным system prompt** | System prompt задаёт роль: «Ты — политический советник собаки породы лабрадор по имени Шарик. Пиши речи в стиле Черчилля, но про косточки и парки» |
| Генерация постов для Instagram/TikTok | **GPT-4o + шаблонизатор Jinja2** | GPT генерирует текст поста, Jinja2 подставляет данные прогулки (километры, фото, погода) в готовый шаблон |
| Голосовые сообщения «от собаки» | **ElevenLabs API (TTS)** | Самый реалистичный Text-to-Speech. Можно создать уникальный «голос» собаки и отправлять владельцу голосовые отчёты в WhatsApp |
| Мультиязычность (для СНГ-рынка) | **DeepL API + GPT-4o** | DeepL — точнее Google Translate для RU/UK/KZ/BY. GPT-4o адаптирует стиль под культурный контекст |

### 💡 Промпт для VIP-тарифа (пример)

```
System: Ты — Шарик, золотистый ретривер, 3 года.
        Ты только что вернулся с прогулки в Парке Горького.
        Пиши торжественную речь в стиле политического лидера,
        но все метафоры — про собачью жизнь.
        Используй пафос, но с юмором.

User:   Шарик прошёл 4.2 км, понюхал 47 столбов,
        облаял 2 голубей и нашёл палку длиной 60 см.
        Погода: солнечно, +18°C.

GPT-4o: «Сограждане! Сегодня я прошёл путь длиной в 4.2 километра —
         путь, который войдёт в историю нашего двора...»
```

## 📦 Полный стек по слоям

| Слой | Технология | Назначение |
|---|---|---|
| **Мобильное приложение** | React Native | Кроссплатформенное приложение iOS/Android |
| **Backend API** | FastAPI (Python) | Быстрый асинхронный REST API |
| **База данных** | PostgreSQL | Основное хранилище данных |
| **Векторная БД** | ChromaDB | Хранение контекста о питомцах для LLM |

| **Облако** | Yandex Cloud / AWS | Хостинг и ML-инференс |
| **Навигация** | OSMnx + NetworkX + ROS 2 | Маршруты и навигация робота |
| **CV** | YOLOv8 + MediaPipe + SAM | Компьютерное зрение |
| **LLM** | GPT-4o + LangChain | Генерация текста и отчётов |
| **TTS** | ElevenLabs | Голосовые сообщения от собаки |
| **Видео** | MoviePy + FFmpeg + Real-ESRGAN | Монтаж TikTok-роликов |
| **Железо робота** | Raspberry Pi 5 + Jetson Nano | Вычисления на борту робота |
| **Сенсоры** | LiDAR + RealSense D435 + GPS | Восприятие окружающей среды |
| **Мониторинг** | Grafana + Prometheus | Мониторинг системы |
| **CI/CD** | GitHub Actions + Docker | Автодеплой и контейнеризация |

## ⚠️ Честные ограничения стека

| Технология | Ограничение | Решение |
|---|---|---|
| GPT-4o API | Платный, ~$0.01 за 1K токенов | На старте — Llama 3 локально |
| ROS 2 + Nav2 | Сложная настройка, крутая кривая обучения | Начать с симулятора Gazebo |
| YOLOv8 (агрессия собак) | Нет готового датасета агрессии | Собрать вручную + аугментация |
| Runway ML Gen-2 | Дорогой генератор видео | На MVP — только MoviePy монтаж |
| Jetson Nano | Дефицит на рынке РФ | Альтернатива — Orange Pi 5 |
