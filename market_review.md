## 🚀 О проекте

**Шарик-шеринг** — концепт инновационного PetTech-сервиса выгула собак,
который объединяет:

- 🤖 **Роботов-ассистентов** для сопровождения прогулок
- 🗺️ **Умные маршруты** с учётом породы, возраста и здоровья питомца
- 🎥 **ИИ-видеоотчёты** — автоматический монтаж хайлайтов каждой прогулки
- 📊 **Аналитику активности** собаки в реальном времени

## 🔍 Анализ конкурентов

### 1. Wag! (США)

> 🌐 Официальный сайт: [wagwalking.com](https://wagwalking.com/)

**Год основания:** 2015  
**Рынок:** США

#### Общие функции
- Выгул собак по запросу (on-demand)
- Передержка и ситтинг
- Дрессировка
- Ветеринарные консультации онлайн

#### Уникальные особенности (УТП)
- **Live GPS-трекинг** — владелец видит маршрут в реальном времени
- **Report Cards** — цифровой отчёт после прогулки (фото, маршрут, данные)
- **Lockbox-сейфы** — умные замки для ключей от квартиры

#### ✅ Плюсы
- Огромная база выгульщиков по всей территории США
- Страхование каждой прогулки
- Бронирование в течение 1 часа
- Высокая прозрачность через GPS

#### ❌ Минусы
- Работает только на рынке США
- Были зафиксированы инциденты с безопасностью собак
- Высокая комиссия с исполнителей (~40%)
- Качество прогулки зависит от конкретного человека

#### 🛠️ Технологии
- GPS / Google Maps API
- Мобильное приложение (iOS / Android)
- Платёжный шлюз Stripe
- Push-уведомления
- Цифровые Report Cards

### 2. Rover (Международный)

> 🌐 Официальный сайт: [rover.com](https://www.rover.com/)

**Год основания:** 2011  
**Рынок:** США, Европа, Канада (10+ стран)

#### Общие функции
- Выгул собак
- Передержка (boarding)
- Дневной уход (doggy daycare)
- Дрессировка

#### Уникальные особенности (УТП)
- **Rover Cards** — отчёты с маршрутом, временем, фото и «индексом настроения»
- **Алгоритмический мэтчинг** — ИИ подбирает идеального выгульщика под
  конкретную собаку
- **Background Check API** — автоматическая проверка личности исполнителей

#### ✅ Плюсы
- Международное присутствие (10+ стран)
- Глубокая система отзывов и рейтингов
- Гибкость услуг (от 30 минут до нескольких недель)
- Верификация исполнителей через ИИ

#### ❌ Минусы
- Качество сильно зависит от конкретного исполнителя
- Нет единого стандарта обучения выгульщиков
- Медленная поддержка в спорных ситуациях
- Модель маркетплейса не гарантирует стабильного качества

#### 🛠️ Технологии
- Machine Learning для мэтчинга «собака ↔ выгульщик»
- NLP для анализа отзывов
- Background Check API
- Облачная инфраструктура
- Мобильное приложение (iOS / Android)

### 3. Собака-Гуляка (Россия)

> 🌐 Официальный сайт: [dog-walk.ru](https://dog-walk.ru/)

**Год основания:** ~2018  
**Рынок:** Россия (крупные города)

#### Общие функции
- Регулярные выгулы собак
- Няня для собак
- Передержка
- GPS-трекинг через приложение

#### Уникальные особенности (УТП)
- **Школа и аттестация выгульщиков** — собственная система обучения
  и проверки качества исполнителей
- **CRM-мониторинг** — менеджеры видят отклонения от маршрута и
  задержки в реальном времени
- Полная адаптация под **российский рынок и законодательство**

#### ✅ Плюсы
- Понимание специфики российского рынка
- Стандартизированное качество (все выгульщики обучены по одной программе)
- Нишевая специализация именно на выгуле
- Человеческий контроль качества через CRM

#### ❌ Минусы
- Ограниченная география (только крупные города РФ)
- Небольшая база исполнителей по сравнению с мировыми игроками
- Ниже уровень технологической экосистемы
- Нет собственного клиентского приложения с полноценным GPS

#### 🛠️ Технологии
- Веб-платформа и онлайн-заказ
- CRM-система (предположительно собственная разработка)
- Геолокация
- Коммуникация с клиентом через мессенджеры

### 4. Tractive (Австрия/Мир)

> 🌐 Официальный сайт: [tractive.com](https://tractive.com/)

**Год основания:** 2012  
**Рынок:** 175 стран

> ⚠️ Tractive — не сервис выгула, а технологическая платформа IoT-трекинга.
> Включён в анализ как **технологическая база** для ИИ-мониторинга
> и геозонирования, которые являются частью концепта Шарик-шеринг.

#### Общие функции
- GPS-трекинг питомцев в реальном времени
- Мониторинг активности и здоровья
- История маршрутов

#### Уникальные особенности (УТП)
- **Wellness Monitoring** — отслеживает фазы сна, калории,
  уровень стресса собаки
- **Virtual Fence (Геозонирование)** — мгновенное уведомление,
  если питомец вышел за пределы безопасной зоны
- Работает в **175 странах** без роуминга

#### ✅ Плюсы
- Не зависит от человека-исполнителя
- Мониторинг здоровья 24/7
- Водонепроницаемые устройства
- Огромный охват по всему миру

#### ❌ Минусы
- Только трекер — не предоставляет услугу выгула
- Требует платной подписки (~5€/мес)
- Зависимость от заряда батареи устройства
- Не анализирует поведение собаки (только физическая активность)

#### 🛠️ Технологии
- GPS + LTE-модуль в ошейнике
- Акселерометр и гироскоп
- Облачная аналитика
- Мобильное приложение (iOS / Android)
- Virtual Fence API

## 🏆 Конкурентные преимущества Шарик-шеринг

### 1. 🎥 ИИ-видеоотчёты вместо обычных фото

Все существующие сервисы присылают владельцу статичные фото и карту маршрута.

**Шарик-шеринг** использует Computer Vision и нейросети для:
- Автоматического монтажа хайлайтов каждой прогулки
- Распознавания поведения собаки (бегала, играла, отдыхала)
- Генерации текстового резюме прогулки через GPT-модели

---

### 2. 🗺️ Умные маршруты на основе ML

Конкуренты предлагают выгул по любому маршруту на усмотрение исполнителя.

**Шарик-шеринг** строит маршрут с учётом:
- Породы, возраста и состояния здоровья питомца
- Прогноза погоды и времени суток
- Загруженности парков и прогулочных зон
- Опасных участков (шумные дороги, стройки, агрессивные зоны)
